⚡ QUICK OPTIMIZATION SETUP GUIDE ⚡

=== STEP 1: Apply Database Indexes (CRITICAL FOR SPEED) ===

Option A - Using phpMyAdmin:
1. Open http://localhost/phpmyadmin
2. Select "adoption_db" database
3. Click "SQL" tab at top
4. Copy and paste all content from optimize_db.sql file
5. Click "Go" to execute

Option B - Using MySQL Command Line:
1. Open Command Prompt (Windows) or Terminal (Mac/Linux)
2. Run: mysql -u root adoption_db < optimize_db.sql
3. Wait for completion

⚠️ IMPORTANT: Skip this step and performance won't improve!

=== STEP 2: Verify .htaccess is Enabled ===

1. Your Apache server must have these modules enabled:
   - mod_deflate (for compression)
   - mod_expires (for caching)
   - mod_headers (for headers)
   - mod_setenvif (for environment)

2. XAMPP Users:
   - Edit C:\xampp\apache\conf\httpd.conf
   - Find lines with #LoadModule and remove the # from:
     * #LoadModule deflate_module modules/mod_deflate.so
     * #LoadModule expires_module modules/mod_expires.so
     * #LoadModule headers_module modules/mod_headers.so
   - Restart Apache

3. To verify: Use Chrome DevTools (F12) > Network > check "Content-Encoding" header should show "gzip"

=== STEP 3: Clear Browser Cache ===

Chrome: Ctrl + Shift + Delete
Firefox: Ctrl + H, then Clear Recent History
Safari: Cmd + Y, then Clear History

=== STEP 4: Test Performance ===

1. Open your website: http://localhost/php/index.php
2. Open Chrome DevTools: F12
3. Go to "Network" tab
4. Reload: F5
5. Check "Total Time" at bottom right
   - Should be 1-2 seconds (much faster than before!)

6. Reload again (2nd time)
   - Should be 0.1-0.5 seconds (cached!)

=== RESULTS YOU SHOULD SEE ===

✅ Page loads 60-95% faster
✅ Smaller file sizes (minified CSS)
✅ Images load on-demand (lazy load)
✅ GZIP compression working

=== IF STILL SLOW ===

1. Verify indexes were applied:
   - phpMyAdmin > adoption_db > right-click table > View > Indexes
   - Should show idx_status, idx_user_id, etc.

2. Check .htaccess:
   - File should exist at C:\xampp\htdocs\php\php\.htaccess
   - Should not have syntax errors

3. Clear server cache:
   - Delete files in: C:\Users\[YourUsername]\AppData\Local\Temp\
     (files starting with "mewmew_cache_")

4. Restart Apache:
   - XAMPP Control Panel > Apache > Stop > Start
